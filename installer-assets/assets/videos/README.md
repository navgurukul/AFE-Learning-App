# Video Assets

This directory will contain video files for lessons.

Video files should be referenced in the content manifest (`manifest.json`) with paths like:
```
"videoUrl": "assets/videos/intro-programming.mp4"
```

## Supported Formats
- MP4 (recommended)
- WebM
- OGG

## Naming Convention
Use descriptive names that match lesson content:
- `intro-programming.mp4`
- `variables.mp4`
- `html-intro.mp4`
- etc.
